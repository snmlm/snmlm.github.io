java基础 return和finally执行问题
发表于2020-07-02|更新于2021-04-24
|阅读量:

执行顺序

  • 当不进入try块时，finally不执行。
  • 当程序还未执行到finally时，程序终止，finally不执行。
  • 其他情况都会执行，在finally中return，那么就会覆盖其他的return。
