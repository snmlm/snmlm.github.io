20 包含min函数的栈
发表于2020-11-16|更新于2020-11-16
|阅读量:

题目描述

  • 定义栈的数据结构，请在该类型中实现一个能够得到栈中所含最小元素的min函数（时间复杂度应为O（1））。

思路

  • 难点在时间复杂度为O(1)上，那么就是min函数直接拿到最小。
  • 另外就是pop操作，假设最小在栈顶，那么pop之后，第二最小值就会变成最小值。
  • 所以应该有个排序之后的集合，跟push和pop同步进行。

    Java 1.8
