二叉树

二叉树遍历

  • 前序遍历：先访问根节点，再前序遍历左子树，再前序遍历有子树
  • 中序遍历：先中序遍历左子树，在访问根节点，再中序遍历右子树
  • 后序遍历：先后序遍历左子树，再后续遍历由子树，再访问根节点
